# 芭芭农场

## 种树小寄巧

### 浇水219次
操作如下：
1. 肥料用到199次(顶部亲密度点开能看到多少次)
2. 淘宝芭芭农场 右上角更多>背包>商城>600肥料买“快速施肥20次”（上限10个，可以囤可以不囤随便）
3. **199+20** 即**219**次

### 速刷浏览任务

:::note 有局限性

适用于大部分浏览会场，但少部分页面无法使用，自测

并且浏览任务所给的肥料也很少（下单占大头），不推荐长期通过此方式刷肥料

:::

操作如下：
1. 将淘宝/支付宝APP打开到芭芭农场，切APP小窗
2. 找到手机时间设置，关闭自动设置，点开调整时间
3. APP进入浏览页面，滑动一次使时间开始计时
4. 将系统时间向后修改1或2分钟
5. 重复3 4步骤
6. 将手机调整回正常时间

### 秒杀砸蛋过肥料

实测黑号也可砸（没任务），也可出0.59商品，按需

操作如下：
1. 淘宝秒杀>1分领商品
2. 砸蛋出的商，原价0.59的商品加购物车
3. 点击[秒杀6W](taobao://pages-fast.m.taobao.com/wow/z/app/ltao-fe/tjb-ssr/home?x-ssr=true&pha_manifest=default&spma=farm&spmb=xsmszywlq&sceneId=7506&deliveryId=68206&itemIds=675098470247)跳转淘宝打开会场
4. 检查列表前几个商品，是否为加车商品（价格对就行）
5. 提交订单，付款
